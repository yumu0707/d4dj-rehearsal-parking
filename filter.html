How to use the filter

  • Checking the "✓" box will display results that contain the selected memeber type.
  • If you have multiple "✓" boxes checked, only results containing ALL selected member types will be displayed. Please refrain from checking more than 4 of these boxes all at once.
  • Checking the "✕" boxes will hide all results that contain the selected memeber type.
  • The following provide the same bonus within each pair:
    • Single match [et4] 3★ and Single match [et2] 4★
    • Both match [et4] 3★ and Both match [et2] 4★
    Please refrain from checking "✓" on one and "✕" on the other within the pair.
  • Please use the "Reset Filter Selection" button if you wish to uncheck any selection.
